# 🎉 بوت قيف أواي عام (Giveaway Bot)

بوت ديسكورد عام يشتغل في **كل السيرفرات** — أي شخص في أي سيرفر فيه البوت يقدر يستخدمه.
مبني بـ **Node.js + discord.js v14**، والتخزين في ملف JSON.

## ✨ المميزات

- `/gcreate` — يفتح **منيو (Modal)** تكتب فيه: الجائزة، عدد الفائزين، المدة.
- زر **🎉 المشاركة** تحت كل قيف أواي (ضغطة وحدة تشارك، ثانية تلغي).
- اختيار الفائزين تلقائياً عند انتهاء الوقت.
- `/glist` عرض القيف أوايات الشغّالة.
- `/gend` إنهاء قيف أواي قبل وقته.
- `/greroll` إعادة سحب فائزين جدد.
- `/ghelp` قائمة الأوامر.
- يكمّل القيف أوايات حتى لو أعدت تشغيل البوت.

## 🛠️ خطوات التشغيل

### 1. إنشاء البوت
1. روح [Discord Developer Portal](https://discord.com/developers/applications).
2. اضغط **New Application** وسمّه.
3. من **Bot** → اضغط **Reset Token** وانسخ التوكن.
4. من **OAuth2 → General** انسخ **Client ID**.

### 2. الإعدادات
انسخ ملف `.env.example` باسم `.env` وحط فيه التوكن والايدي:
```
DISCORD_TOKEN=التوكن
CLIENT_ID=الايدي
```

### 3. التثبيت
```bash
npm install
```

### 4. تسجيل الأوامر (مرة وحدة، أو كل ما تضيف أمر)
```bash
npm run deploy
```

### 5. تشغيل البوت
```bash
npm start
```

### 6. دعوة البوت لأي سيرفر
استبدل `CLIENT_ID` برقم بوتك وافتح الرابط:
```
https://discord.com/api/oauth2/authorize?client_id=CLIENT_ID&permissions=2147485696&scope=bot%20applications.commands
```
الصلاحيات: إرسال رسائل + إدارة الرسائل + استخدام أوامر السلاش.

## 📌 ملاحظات
- التسجيل العالمي للأوامر ممكن ياخذ لين ساعة عشان يظهر في كل السيرفرات.
- أوامر الإدارة (`gcreate`, `gend`, `greroll`) تحتاج صلاحية **Manage Server**.
- البيانات تتخزن في `data/giveaways.json`.
